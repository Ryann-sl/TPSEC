HOW BRUTE FORCE WORKS
=====================

Brute force is a method used to find a password by trying every single possible combination of characters until the right one is found.

Heres is how the code I wrote works step-by-step:

1. THE CHARSET (The Ingredients)
   First, we decide which characters we are allowed to use. 
   - For a simple numeric PIN, we use "0123456789".
   - For a complex password, we use Letters + Numbers + Symbols.

2. THE COMBINATIONS (The Generator)
   The code uses a tool called `itertools.product`. 
   If our length is 2 and our charset is "12", it will create:
   "11", "12", "21", "22"
   It does this for EVERY possible sequence.

3. THE LOOP (The Guessing)
   We start a loop that takes each generated combination one by one:
   - It turns the combination into a string (e.g., "123").
   - It checks: Is this the same as the target password?
   - If YES: We stop and say "SUCCESS!".
   - If NO: We move to the next combination.

4. THE STORE (Session Management)
   We use a 'sessions' dictionary to keep track of the attack. 
   This allows the user to PAUSE or STOP the loop at any time. 
   The loop checks the 'sessions' store at every single step to see if it should keep going or wait.

5. THE SPEED
   Brute force can take a very long time if the password is long or the charset is large. 
   That's why we log the progress (e.g., "Attempt 100,000") so the user knows the computer is still working!
